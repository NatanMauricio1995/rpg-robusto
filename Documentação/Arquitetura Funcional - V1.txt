================================================================================
SISTEMA RPG — ARQUITETURA FUNCIONAL DEFINITIVA v1.0
================================================================================

Filosofia do Sistema

O sistema é baseado em:
  - D&D 5e Adaptado

Princípios:
  - ✓ Automatizar cálculos
  - ✓ Automatizar progressão
  - ✓ Automatizar recursos
  - ✓ Automatizar descanso
  - ✓ Automatizar subida de nível
  - ✓ NÃO automatizar narrativa
  - ✓ NÃO automatizar decisões do mestre
  - ✓ NÃO automatizar combate completo

================================================================================
TIPOS DE USUÁRIO
================================================================================

Mestre
Pode:
  - Criar campanhas
  - Criar mundos
  - Criar biblioteca RPG
  - Editar tudo
  - Controlar sessões
  - Aprovar personagens
  - Editar fichas

Jogador
Pode:
  - Criar personagens
  - Editar seus personagens
  - Visualizar biblioteca
  - Participar de campanhas

Não pode:
  - Criar campanhas
  - Editar biblioteca
  - Excluir componentes globais

FICHA DE CAMPANHA
================================================================================

Cada personagem pode participar de várias campanhas.
Cada campanha possui uma ficha independente.

Possui:
  - Campanha
  - Personagem
  - Nível
  - XP
  - HP Atual
  - HP Máximo
  - Inventário
  - Equipamentos
  - Moedas
  - Habilidades
  - Magias
  - Recursos

================================================================================
MOEDAS
================================================================================

Sistema fixo.
  - Cobre
  - Prata
  - Ouro
  - Platina

Conversão:
  - 10 Cobre = 1 Prata
  - 10 Prata = 1 Ouro
  - 10 Ouro = 1 Platina
