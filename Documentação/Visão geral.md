**Objetivo**: explicar o que o sistema é.

## 1. Objetivo
Sistema de gerenciamento de RPG de mesa.

**Foco:** 
- Auxiliar mestres e jogadores. 
- Não automatizar narrativa.
- Não substituir o mestre.

## 2. Escopo
- Cadastro de mundos
- Cadastro de biblioteca RPG
- Cadastro de personagens
- Gerenciamento de campanhas
- Gerenciamento de combates
- Relatórios

## 3. Tecnologias
- Next.js
- React
- Firebase Auth
- Firestore
- Firebase Storage

## 4. Perfis
- Mestre
- Jogador

## 5. Princípios
- Desktop First
- Componentização
- D&D 5e Adaptado
- Progressão Automática
- Combate Assistido
